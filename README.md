# Weather-Forecast-Desktop



🌦️ Weather Forecast Desktop

A modern and responsive weather forecast web application that provides real-time weather updates for any location. Built using HTML, CSS, and JavaScript with a clean UI and smooth user experience.

## 🚀 Features

- 🌍 Search weather by city name  
- 🌡️ Real-time temperature, humidity, and wind speed  
- 🎨 Dynamic UI based on weather conditions  
- ⚡ Fast and responsive design  
- 📱 Fully responsive (Mobile, Tablet, Desktop)  
- 🔔 Alert for high temperature (above 40°C)  
- 🎬 Smooth animations and modern UI  

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Tailwind CSS (via CDN)  
- Font Awesome Icons
